# Next.js× ホットペッパーグルメ API でグルメ検索アプリ制作

## 機能

- 地域別検索
- ジャンル別検索
- キーワード検索
- ページネーション
- PWA 対応

## 参考

- [ホットペッパー API リファレンス](https://webservice.recruit.co.jp/doc/hotpepper/reference.html)
- [Hotpepper API を使って指定した地点の近くの飲食店を検索する](https://coffee-nominagara.com/hotpepper-api)
